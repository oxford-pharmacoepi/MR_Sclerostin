# Meta-analysis related files
1. preprocessing.R
It reads the three gwas and prepares the data for the meta-analysis.
It generates the files:
 - ng1_NP.txt: GWAS from Nature. The file includes the SNPs from chromosome 17 only.
 - ng2_NP.txt: GWAS from Nature. The file includes the SNPs from chromosome 17 only.
 - ng.txt: Includes only the SNPs that have passed some "quality checks" by me. Is the file that will be used in the meta-analysis.
 - sc_NP.txt: GWAS from Science. The file includes the SNPs from chromosome 17 only.
 - sc.txt: Includes only the SNPs that have passed some "quality checks" by me. Is the file that will be used in the meta-analysis.
 - nn_NP.txt: GWAS from Nature Genetics. The file includes the SNPs from chromosome 17 only.
 - nn.txt: Includes only the SNPs that have passed some "quality checks" by me. Is the file that will be used in the meta-analysis.
2. GWAMA
It performs the fixed effect/random effect methods for the meta-analysis.
The file: gwama.in is needed to run this program (more instructions, see processing.R file)
The generated files are stored in the "Fixed_results" and "Random_results" directories.
3. METAL
It performs the fixed effect method for the meta-analysis. There are outcomes that GWAMA does not extract, so both programs are used to obtain
to obtain all the required information.
It generates the METAANALYSIS1.TBL
3. processing.R
It reads the files generated by GWAMA and METAL and converts them into .csv. It stores the results in the Fixed_results and Random_results directory
It also creates the file Fixed_vs_random.csv, which is a table used to compare both analysis.
